# Game Timer

A lightweight, browser-based game timer for turn-based games.

## Features
- Setup screen for player count, names, and parameters
- Big next-player button
- Pause/resume
- Manual time adjustments
- Optional individual pause buttons

## Run locally
Open index.html in your browser.
